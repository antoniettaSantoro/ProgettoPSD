# ProgettoPSD
**Esecuzione del Gestionale:**  
Posizionarsi nella radice.

```bash
./segnalazioni
```

**Esecuzione dei Test:**  
Posizionarsi nella radice.
```bash
./run_tests
```

### Compilazione  
Makefile realizzato in Linux.  
Digitare `make` nella radice per compilare sia 'segnalazioni', che 'run_tests'

---

### Struttura
- **/include**: File header (.h)
- **/src**: Implementazione (.c)
- **/tests**: Casi di test ( */test_cases*) e implementazione dei test
- **/txt_files**: File report, risultati dei test e file contenenti dati del programma salvati e/o da caricare
